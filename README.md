# Hexagon Map Tool V2 - Documentación

Documentación oficial y guía visual para el uso de **Hexagon Map Tool V2**, una herramienta avanzada para la creación y edición de mapas hexagonales en Unity.

## ¿Qué es Hexagon Map Tool V2?
Herramienta modular para Unity que permite diseñar, pintar y gestionar mapas hexagonales 3D de forma eficiente, con soporte para múltiples capas, pinceles personalizables, patrones de pintado y gestión avanzada de objetos.

## Características principales
- Interfaz intuitiva y modular
- Pintado masivo de hexágonos y props
- Configuración avanzada de grids y capas
- Paletas de pinceles personalizables
- Importación/exportación de capas en JSON
- Soporte multilenguaje (Español, Inglés, Japonés)
- Personalización visual y de atajos
- Herramientas de debug y solución de conflictos
- Integración sencilla de modelos 3D

## Estructura de la documentación
- **index.html**: Documentación principal, guía visual y referencia completa
- **css/**: Estilos visuales personalizados
- **js/**: Scripts para navegación y experiencia de usuario
- **img/**: Imágenes, diagramas y ejemplos visuales

## Uso
1. Abre `index.html` en tu navegador para acceder a la documentación interactiva.
2. Navega por las pestañas y secciones para consultar guías rápidas, referencia de componentes, configuración avanzada y solución de problemas.
3. Utiliza los enlaces de comunidad para soporte, tutoriales y contacto directo.
